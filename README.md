# Frontend Practice

学習用のフロントエンド練習リポジトリです。  
HTML / CSS / JavaScript を使用して以下の機能を実装しました：  
- 自己紹介ページ  
- セクション展開機能  
- トップへ戻るボタン  

---
## How to Run
ローカルで `index.html` をブラウザで開くだけで実行できます。

## プロジェクト構成
- **test5.html** : メインページ（ポートフォリオページ）
- **style5.css** : ページのスタイル定義
- **app.js** : ページに動きを与えるJavaScript
